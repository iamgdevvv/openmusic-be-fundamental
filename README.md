We have provided Postman Collection and Environment files that you can use for testing. Please download the file at the following link: [Postman OpenMusic API V1 Test Collection dan Environment](https://github.com/dicodingacademy/a271-backend-menengah-labs/raw/099-shared-files/03-submission-content/01-open-music-api-v1/OpenMusic%20API%20V1%20Test.zip)
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

list things you need to use the software and how to install them.
-   postgresql
-   node
-   npm

### Installing and Running

```bash
$ psql --username postgres
postgres=# CREATE USER postgres_om WITH PASSWORD '0okm9ijn8uhb';
postgres=# ALTER USER postgres_om CREATEDB;
postgres=# \q

# Clone this repository
$ git clone https://github.com/iamgdevvv/openmusic-be-fundamental

# Go into the repository
$ cd openmusic-be-fundamental
$ npm install
$ npm run start::dev
```
